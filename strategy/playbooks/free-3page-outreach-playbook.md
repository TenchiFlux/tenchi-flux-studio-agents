# Free 3-Page Website Outreach Playbook (Tenchi Flux Studio)

Date: 2026-03-08
Purpose: reusable outreach framework based on first successful sends (Jeramy Jackson + Hyder family).

## What worked in this message style

1. Warm personal reconnect opener.
2. Honest personal context (year off coaching / mental reset).
3. Credibility anchor (current web/content work).
4. Clear offer: free simple 3-page website.
5. Clear timeline: rough draft in ~1 week.
6. Clear value: publish + local Google visibility.
7. Soft ask: social shoutout + referrals.
8. Transparent reason: portfolio/case study build before public launch.

## Reusable template v1

"[Name]! Been a minute. Quick update: I’ve been deep in web work and building simple sites for local businesses. I’m expanding my portfolio right now — if you don’t already have a site, want a **free 3-page website** for [business name]?

I can send a rough draft in about a week, then finalize and publish it live.

No fee — just a social shoutout and referrals if you love it. I’m building case studies ahead of my public launch this summer. Open to it?"

## 3 personalization slots

- Relationship hook: friend/family/context callback.
- Business-specific trigger: recent post, new location, service launch, etc.
- Credibility line: current role + web execution proof.

## Fast follow-up sequence

- Day 0: initial outreach.
- Day 2: soft bump ("totally fine if timing’s off").
- Day 5: close-the-loop note ("want me to circle back next month?").
