# Tenchi Flux Studio Agents

A curated, production-focused fork of [`msitarzewski/agency-agents`](https://github.com/msitarzewski/agency-agents), tailored for agency delivery workflows.

- Source license: **MIT**
- Original project: The Agency (full role library)
- This repo focus: **market agency execution** (strategy → production → QA → reporting)

## Why this fork exists

The original repo is broad (many roles across many domains). This fork narrows scope to the agents that map directly to Tenchi Flux Studio services.

## What was removed in curation

- Spatial-computing division
- Non-core roles not directly tied to agency delivery
- Experimental breadth roles outside current Tenchi Flux launch scope

## Suggested operating model

1. **Orchestrator** defines mission and acceptance criteria.
2. Specialist agents produce artifacts to agreed contracts.
3. Testing agents gate quality with proof.
4. Support/reporting agents produce client-ready summaries.
